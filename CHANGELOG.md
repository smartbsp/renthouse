# 工作紀錄

## 2026-05-22

### 建立電費分攤 App（rentHouse）
- 建立 PWA 電費分攤計算機 `C:\Users\user\Desktop\renthouse\`
  - `index.html`、`manifest.json`、`sw.js`、圖示
- 功能：兩戶（11F-1 戶A / 11F-2 戶B）電費分攤計算
- 分電表：兩戶各自輸入上期/本期分表度數，用電量自動結算
- 總用電度數 = 戶A + 戶B 自動加總
- 每度單價 = (總電費 − 公電 − 底度) / 總度數
- 各戶電費 = (用電度數 × 單價) + 公電分攤(÷2)
- 支援歷史紀錄儲存、詳情彈窗、草稿自動暫存
- 基本底度費預設 40 元
- 選起日自動推算訖日（+2 月）
- 選起日自動帶入上一期的分表度數
- 以 `renthouseRecords` 為 localStorage key

## 2026-05-22 後續優化

### 台電帳單擴充
- 新增「計費度數」「本次抄表日」「下次抄表日」欄位
- 本次抄表日 = 訖日 + 1 日，下次抄表日 = 本次抄表日 + 2 個月 + 1 日（可手改）
- 新增「當期每度平均電價」= 總電費 ÷ 計費度數，提示上期數值
- 新增「帳單月份」欄位（唯讀，自動從訖日推算民國年月）
- 計費度數步進改為 10 度

### 分電表擴充
- 新增「估算公電度數」= 計費度數 − 總用電度數
- 新增「上一期」「下一期」導覽按鈕，按訖日排序瀏覽歷史紀錄

### 歷史紀錄強化
- 搜尋：月份、期間雙下拉選單 + 快速篩選
- 排序：點擊欄位標題升冪/降冪切換，▲▼ 箭頭標示
- 編輯：✎ 按鈕載入表單，✕ 刪除（確認防呆）
- 歷史表格新增「總電費」欄
- 清除全部：雙重確認 + 自動備份至 `renthouseBackup`

### 批次輸入
- 獨立卡片，表格式輸入月份、起訖日、抄表日、AB 上期/本期度數
- 輸入抄表日自動推算起訖日與月份
- 起日 = 上一期抄表日（從歷史紀錄匹配），上期度數自動帶入
- 按 Enter 跳下一欄位
- 批次儲存時檢查重複月份，列出跳過清單
